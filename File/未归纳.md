### 配置GitServer

> SSH

```git
# 创建一个专门的用户
useradd git
# 设置密码
passwd 密码
# 初始化空仓库
git init --bare --shared
# 克隆远程仓
git clone user@git.example.com:/opt/git/my_project.git
# 生成ssh密钥
ssh-keygen
# 给git文件夹权限
sudo chown -R git:git sample.git
# 试一下吧！
```
